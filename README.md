# Poop Map - 便便地圖

一個多語言便便追蹤應用程式，支援 Google 登入和 Google Maps 整合。

## 下載應用

### Android 版本
- 最新版本: v1.0.0 (2025-11-03)
- 下載鏈接: [poop-map-v1.0.0-debug.apk](releases/poop-map-v1.0.0-debug.apk) (9.93 MB)
- 最低要求: Android 7.0+
- 詳細說明: [發布說明](releases/README.md)

### 其他平台
- iOS: 開發中
- Web 版: [https://poop-map.vercel.app](https://poop-map.vercel.app)

## 功能介紹

### 核心功能
- 支援 8 種語言：英語、繁體中文、簡體中文、日語、韓語、西班牙語、法語、德語
- Google 帳號登入
- Google Maps 地圖顯示
- 記錄便便位置
- 隱私設定：公開、好友、私人

### 社交功能
- 好友系統
- 通知中心
- 挑戰系統
- 動態牆
- 成就系統
- 排行榜

### 道具系統
- 收集便便道具
- 使用道具效果
- 管理道具庫存

### 原生功能（Android）
- 相機拍照
- 分享功能
- 觸覺反饋
- 離線支持

## 使用說明

### 1. 安裝應用

#### Android 設備
1. 下載 APK 文件
2. 在設備上啟用「允許安裝未知來源」
3. 點擊 APK 文件進行安裝
4. 允許必要的權限（位置、相機、存儲）

#### Web 版本
直接訪問 [https://poop-map.vercel.app](https://poop-map.vercel.app)

### 2. 首次使用

#### 登入
1. 打開應用
2. 點擊「使用 Google 登入」按鈕
3. 選擇你的 Google 帳號
4. 允許應用訪問你的基本資料

#### 設置個人資料
1. 登入後，點擊右上角的用戶頭像
2. 可以修改顯示名稱（每人可免費改名一次）
3. 查看個人統計資料

### 3. 記錄便便

#### 基本記錄
1. 確保已允許位置權限
2. 點擊畫面下方的「拉一坨」按鈕
3. 應用會自動獲取當前位置
4. 選擇便便類型（可選）
5. 添加描述（可選）
6. 選擇隱私設定：
   - 公開：所有人可見
   - 好友：僅好友可見
   - 私人：僅自己可見
7. 點擊確認完成記錄

#### 拍照記錄
1. 在記錄便便時，點擊相機圖標
2. 拍攝照片或從相冊選擇
3. 照片會與便便記錄關聯

### 4. 查看地圖

#### 瀏覽便便
1. 地圖上會顯示不同顏色的便便標記：
   - 綠色：你的便便
   - 藍色：好友的便便
   - 橙色：公開的便便
2. 點擊標記查看詳細資訊
3. 可以縮放和拖動地圖

#### 篩選顯示
1. 點擊左上角的篩選按鈕
2. 選擇要顯示的便便類型：
   - 全部
   - 我的
   - 好友的
   - 公開的

### 5. 社交功能

#### 添加好友
1. 點擊右上角的好友圖標
2. 搜索好友的 email 或名稱
3. 發送好友請求
4. 等待對方接受

#### 查看通知
1. 點擊右上角的通知圖標
2. 查看好友請求、挑戰邀請等通知
3. 點擊通知進行相應操作

#### 參與挑戰
1. 點擊挑戰圖標
2. 查看當前活動的挑戰
3. 點擊「參加」加入挑戰
4. 完成挑戰目標獲得獎勵

#### 查看排行榜
1. 點擊排行榜圖標
2. 選擇時間範圍：
   - 本週
   - 本月
   - 全部時間
3. 查看自己和好友的排名

### 6. 道具系統

#### 獲得道具
1. 記錄便便時有機會獲得道具
2. 完成挑戰獲得道具
3. 解鎖成就獲得道具

#### 使用道具
1. 點擊右下角的道具庫存圖標
2. 查看擁有的道具
3. 選擇道具並點擊使用
4. 道具效果會立即生效

### 7. 成就系統

#### 查看成就
1. 點擊成就圖標
2. 查看已解鎖和未解鎖的成就
3. 成就分為三類：
   - 數量成就：記錄便便數量
   - 品質成就：便便品質相關
   - 特殊成就：特殊條件達成

#### 解鎖成就
1. 完成特定條件自動解鎖
2. 解鎖後獲得成就點數
3. 某些成就會獎勵道具

### 8. 設定

#### 語言切換
1. 點擊右上角的語言按鈕
2. 選擇你想要的語言
3. 界面會立即切換

#### 音效設定
1. 點擊設定圖標
2. 開啟或關閉音效
3. 設定會自動保存

#### 隱私設定
1. 在個人資料中設定默認隱私級別
2. 每次記錄時可以單獨調整

### 9. 常見問題

#### 無法獲取位置
1. 檢查是否已允許位置權限
2. 確保 GPS 已開啟
3. 在室外或窗邊獲得更好的信號

#### 登入失敗
1. 檢查網路連接
2. 確保設備已安裝 Google Play Services（Android）
3. 嘗試重新啟動應用

#### 地圖無法載入
1. 檢查網路連接
2. 確保有足夠的數據流量
3. 嘗試重新整理頁面

#### 道具無法使用
1. 檢查道具是否已過期
2. 確認是否符合使用條件
3. 重新啟動應用後再試

### 10. 注意事項

#### 權限說明
- 位置權限：用於記錄便便位置
- 相機權限：用於拍攝便便照片
- 存儲權限：用於保存照片
- 震動權限：用於觸覺反饋

#### 數據使用
- 應用需要網路連接才能同步數據
- 地圖載入會消耗數據流量
- 建議在 WiFi 環境下使用

#### 隱私保護
- 你的位置數據僅用於記錄便便
- 可以選擇隱私級別控制誰能看到你的記錄
- 隨時可以刪除自己的記錄
