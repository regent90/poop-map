## 🚀 快速開始

### 開發環境設置
```bash
# 安裝依賴
npm install

# 啟動開發服務器
npm run dev

# 構建移動應用
npm run cap:build

# 運行 Android 應用
npm run cap:run:android
```

### 主要技術棧
- **前端**: React + TypeScript + Tailwind CSS
- **移動端**: Capacitor
- **後端**: Convex
- **地圖**: Google Maps API
- **認證**: Google OAuth 2.0
